# pimm

## pip mirrors manager

This is a pip mirrors manager, just like [nrm](https://github.com/Pana/nrm) for npm.

### support command

```bash
pmm ls # list all server
pmm help # show help
pmm test # show response time of all servers 
pmm use <server name>
pmm -V # show current version
```
